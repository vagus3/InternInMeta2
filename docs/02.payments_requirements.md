# 결제 모듈 요구사항 분석 및 기능 목록

## 📌 기본 정보
### 프로젝트명: 
예) 패션 쇼핑몰 결제 모듈

### 사용하게 될 기술: 
React, React Router

### 문서 작성자: 
심형규

## 📝 고객 요구사항 정리
- 사용자가 카드 정보 입력 후 카드 저장하기
- 카드 정보 입력 시 cvc, 비밀번호 등 개인정보 보안 강화
- 입력한 정보 중 카드 번호 뒷8자리는 *로 표시
- 카드 추가 시 밑에 카드를 선택해서 결제할 수 있도록 결제하기 버튼
- 입력한 정보 이미지 상에서 확인할 수 있게 하기
