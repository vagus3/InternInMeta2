# 결제 모듈 리뷰

## 고객사 요구사항
1. 카드 정보 입력 했을 경우 위 카드 이미지에 입력한 정보 표기하기
2. 위 1번 개발 과정에서 앞에서부터 8자리까지만 표기될 수 있도록 하기
3. 보안코드에 관한 설명을 물음표 아이콘에 커서를 가져다대거나 혹은 누를 시 모달 창 띄우기
4. 카드 소유자 이름 작성 시 해당 문자가 얼마나 쓰였는지 이름 입력 창 오른쪽 상단에 표기
5. 카드가 없을 때는 새로운 카드를 추가해주세요 라는 문구를 띄우고 등록 된 카드가 있을 경우 해당 문구는 사라지고 등록된 카드 밑에 결제하기 버튼 추가(새로운 카드 등록 칸은 유지)
6. 카드 이미지에 그림자 추가
7. 입력 창에 작성해야할 내용 예시 띄워놓기
