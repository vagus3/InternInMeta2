# 장바구니 페이지 요구사항 분석 및 기능 목록

## 📌 기본 정보
### 프로젝트명: 
장바구니 결제 페이지 구성

### 사용하게 될 기술: 
React

### 문서 작성자: 
심형규, 최대리

## 📝 고객 요구사항 정리

●	금액이 정확하게 보일 수 있게 구성
●	총 결제 금액이 10만원 이상이면 배달비 무료, 아닐 시 배송비 3천원 발생임을 사용자가 알 수 있도록 표기
●	뒤로가기 버튼을 누를 시 메인 상품 목록 페이지로 이동할 수 있도록 구성
●	결제하기 버튼을 눌렀을 때, 결제 모듈을 모달로 사용할 수 있게 작성
