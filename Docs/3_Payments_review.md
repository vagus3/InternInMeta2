# 결제 모듈 리뷰

## 고객사 요구사항
장바구니 페이지에 총 상품 금액을 밑에 표기하고 그 밑에는 배송비, 총 결제 금액 순으로 구성 
맨 밑에 결제하기 버튼을 구성
총 상품 금액의 액수에 따라 배송비를 다르게 표기(10만원 이상일 때는 배송비 무료 혹은 0원 미만일 경우는 3,000원 표기)
뒤로가기 버튼 추가


## 요구사항에 따른 기능 동작 확인

상품 목록에 +, - 버튼을 통해 품목 수량을 추가했을 때 맨위 총 수량 개수 및 각 상품 개수 잘 올라가는지 확인
위 내용을 진행했을 때 총 상품 금액에 잘 반영되고 배송비의 표기가 잘 변경 되는지 확인
