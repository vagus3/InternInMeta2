# 결제 모듈 요구사항 분석 및 기능 목록

## 📌 기본 정보
### 프로젝트명: 
패션 쇼핑몰 경제 모듈 구성하기

### 사용하게 될 기술: 
React

### 문서 작성자: 
예) 심형규, 최대리

## 📝 고객 요구사항 정리
1. 카드 등록 시 여러 개의 카드가 등록될 수 있게 구현
2. 카드 등록 과정에서 민감한 정보(보안코드, 비밀번호 등등)는 정보 등록 시 보이지 않게 하기
3. 결제 모듈의 경우 상품 목록 담기 버튼 옆에 '결제' 버튼을 추가해서 동작할 수 있게 하기
## 📋 기능 목록
